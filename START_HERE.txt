COSA È STATO CREATO?
═══════════════════════════════════════════════════════════════════════════════

Una web app mobile che fotografa dipinti e li analizza con AI (Claude).
Ottiene: nome, artista, significato, ubicazione, materiali, ecc.

ARCHITETTURA:
──────────────────────────────────────────────────────────────────────────────

  Frontend (HTML/CSS/JS)  →  Backend (Node.js)  →  AI (Claude API)

  📱 Fotocamera          📡 API Express        🤖 Analisi Vision
  🖼️ Visualizzazione      📊 Gestione richieste   🎨 Riconoscimento arte


AVVIARE IN 3 STEP:
═══════════════════════════════════════════════════════════════════════════════

1. OTTIENI API KEY (gratuita!)
   └─ https://console.anthropic.com/account/keys
   └─ Clicca "Create Key" e copia la chiave (sk-ant-...)

2. CONFIGURA .env
   └─ Apri file ".env"
   └─ Sostituisci: ANTHROPIC_API_KEY=sk-ant-your-key-here

3. AVVIA BACKEND
   └─ Terminal: cd backend && npm install && npm start
   └─ Apri: http://localhost:3000 nel browser
   └─ Scatta foto → Analizza → Vedi risultati!

TROUBLESHOOTING VELOCE:
═══════════════════════════════════════════════════════════════════════════════

ERRORE: "Module not found: express"
  └─ Soluzione: cd backend && npm install

ERRORE: "No module named anthropic"
  └─ Soluzione: pip install anthropic

ERRORE: "ANTHROPIC_API_KEY not found"
  └─ Soluzione: Controlla .env (deve avere la chiave reale)

ERRORE: "Port 3000 already in use"
  └─ Soluzione: Cambia PORT in .env in un'altra porta (es: 3001)

ERRORE: "Camera permission denied"
  └─ Soluzione: Consenti l'accesso alla camera nel browser
