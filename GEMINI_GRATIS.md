## 📝 Come Ottenere l'API Key (3 minuti)

### 1. Vai al sito ufficiale
👉 https://aistudio.google.com/apikey

### 2. Clicca "Create API Key"
- Seleziona "Create API key in new project"
- Google crea la chiave automaticamente

### 3. Copia la chiave
Sarà qualcosa come: `AIzaSyD...`

### 4. Incollala nel file `.env`
Apri `.env` e sostituisci:
```
GEMINI_API_KEY=AIzaSyD_your_actual_key_here
PORT=3000
```

## 🚀 Avvia l'App (1 minuto)

```bash
# Installa dipendenze Google
pip install google-generativeai
```
